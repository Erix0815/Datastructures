Makefile

target usage
demo build and run the current demo
test build and execute all unittests
build build the library
doc generate documantation
fmt format all code
clean delete files ignored by git
install build and install the library
remove uninstall the library

Usage

After installing the library you can include it using #include <mostly-trees/mostly-trees.hpp>.\ Dont forget to also link the library when compiling (e.g.: -l mostly-trees).
